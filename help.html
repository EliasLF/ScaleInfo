The Scale Explorer

In this section you can search for scales, for which you can then display playing instructions for various instruments and general information about the scale in the other sections. To find the scale you are looking for you can set filters.

In the top half of the Scale Explorer there are 3 main filters to choose from:
  • With "Scale Degrees" the algorithm compares the entered degrees by the user with the scale degrees of the scales in the data base. At this 1 corresponds to the root, 1#/2b to the note 1 half step above the root, 2 to the note 2 halfsteps above the root and so forth. The desired root note can be set at the drop-down menu below the degree buttons. Below that is a button, which enables you to set the entered scale instantly without searching for it in the result list.
    This filter is useful if you already know where the root of the scale is, for example if you have a chord and want to find a suitable scale to improvise over it. In this case you would enter the degrees of the chord notes, choose the same root note in the drop-down menu as the chord has and allow more added notes in the input field below the main filters.
  • With "Scale Notes" you can set notes and the algorithm will find all scales with the appropriate root note for each.
    This is useful if you just know the notes of a scale and not which of them is the root.
  • The last option "All" just shows all scales in the database with the root note being adjusted in the drop-down menu below.

Below the main filters there are a few extra filters which narrow down or widen the results:
  • The first 2 number input fields add results by allowing notes to be added to the user input or to be altered (which means changing a note 1 half step up or down).
  • With "Scale Category" you can limit the search to one or multiple specific types of scales (mainly by origin).
  • The last filter is a search bar, which restricts the search to results, which contain the search phrase.

All of these filters are followed by the result box displaying all scales found in the database matching the filter settings.

With the "I'm feeling lucky"-button below the result box you can explore the database by randomly choosing scales from the result box.(Note that this button only picks from the results set by the filters. If you want to choose a random scale from the whole database you have to remove all filter restrictions, i.e. set the main filter to "All", select all scale categories and empty the search box.)
